집필중인 파일을 만들기!

hexo new draft “파일명”

이렇게 작성한 파일은 작성중인 것으로 보이지 않음!!

hexo sever –draft로 하면

작성중인 파일들을 볼 수 있음

집필이 끝난 후엔

hexo publish 파일명

을 하게되면 draft에 있던 파일이 post로 이동하고,
hexo server를 하게되도 파일이 보여지게 됨!!

주의할 점은 hexo로 설치한 폴더를 지울 경우 복구할 수 가 없음!!
Github에는 소스 코드 전체가 올라간 것이 아닌 public에 있는 것들만 올라갔기 때문임!

myBlog라고 폴더 전체를 올려놓은 repository를 하나 만드는 것으로 문제를 해결할 수 있음!
